# /fresh Report Template

After reading this report, the user should be able to say "OK, let's do this" without reading anything else.
No empty statements. Be specific.

```markdown
## Last Session Summary

{What was done last session. Key points only, 2-3 lines.}

## What To Do Now

1. {Most urgent task}
2. {Next}
3. {Next}

## Points of Attention

{Blockers, stale warnings, conflicts, etc. if any. If none, omit this section entirely.}

## Pro Insights

{If there was a GPT Pro response, summarize key insights here. If not, omit this section entirely.}
```

Do not leave empty sections as blank — remove them entirely. Do not waste the user's time with unnecessary information.
