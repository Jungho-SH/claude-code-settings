# exp_log.md 작성 템플릿

로그 없는 실험은 안 한 거다.

```markdown
## YYYY-MM-DD
### 실험명

- **서버/경로**: {어디서 돌렸는지}
- **의도**: 왜 이 실험을 돌렸는지. 가설이 뭐였는지.
- **설정**:
  - 시드: {N}
  - 주요 파라미터: {key=value}
  - 변경점: 이전 실험 대비 뭐가 달라졌는지
- **결과**:
  - 숫자: {metric=value}
  - 관찰: {그래프/영상에서 본 것}
- **결론**: 뭘 알게 됐는지. 가설이 맞았는지 틀렸는지.
- **다음**: 이 결과를 바탕으로 다음에 뭘 할 건지.
```
